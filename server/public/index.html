Instructions

The Stroop Task is a psychological experiment named after John Ridley Stroop. The Stroop phenomenon demonstrates the difficulty of matching the ink color of a color word if there is a mismatch between ink color and word. For example, the word GREEN printed in red ink.

In the following tasks, you will see colored words (like GREEN, or BLUE). You need to respond to the color of the words (not the meaning) by pressing the key 'f' if the color and word match and the key 'j' if there is a mismatch:

  • Match ('f'): The color of the word and the meaning is the same (e.g., GREEN)
  • Mismatch ('j'): The color of the word and the meaning is not the same (e.g., GREEN)
